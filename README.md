## gitでブランチを作る

ブランチとは技のこと、開発において違う継投で行いたい場合はブランチを作ります。
かっこよくいうとブランチを切るといいます。

- 今のブランチを見る
'''
git branch
'''

-　ブランチを作成する
'''
git branch test_branch
'''

- ブランチを移動
'''
git checkout test_branch
'''
マジだるい
